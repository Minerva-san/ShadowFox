# 🛍️ AJIO UX+

An enhanced e-commerce frontend inspired by AJIO, focused on improving user experience through modern UI, interactive features, and seamless shopping workflows.

🔗 **Live Demo:** https://minerva-san.github.io/ShadowFox/

---

## 📌 Project Overview

AJIO UX+ is a frontend enhancement project that recreates and improves the shopping experience with dynamic product discovery, persistent cart and wishlist management, advanced filtering, smart search, and a responsive checkout flow.

The project emphasizes clean UI/UX design and frontend functionality using HTML, CSS, Bootstrap, and JavaScript.

---

## ✨ Features

### 🛒 Shopping Experience
- Product listing page
- Browse products by category
- Real AJIO product links
- Quick View product modal
- Recently Viewed products

### 🔍 Smart Product Discovery
- Live search
- Search suggestions
- Category filtering
- Price filtering
- Rating filtering
- Multiple sorting options:
  - Most Popular
  - Trending
  - Best Seller
  - Price: Low → High
  - Price: High → Low
  - Highest Rated
  - Highest Discount
  - Newest

### ❤️ Wishlist
- Add/Remove wishlist items
- Persistent wishlist using LocalStorage
- Wishlist page
- Move items directly to cart

### 🛍️ Shopping Cart
- Add to Cart
- Live cart preview
- Remove products
- Persistent cart using LocalStorage

### 💳 Checkout
- Delivery address form
- Payment method selection
- Order summary
- Order confirmation popup

### 📱 Responsive Design
- Mobile-friendly layout
- Bootstrap components
- Responsive navigation
- Responsive product grid
